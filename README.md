# 🏎️ F1 Dashboard

A Formula 1 season tracker built entirely through vibe coding — not a single line written manually. Stack decisions were made by me, code was generated by [Claude](https://claude.ai/code).

> _"I chose Python because that's where my debugging skills live. Everything else? Vibes."_

---

## What it does

- **Season overview** — live driver & constructor standings pulled straight from the F1 API
- **Championship progression** — interactive points-over-rounds chart, toggle between drivers and constructors
- **Teammate H2H** — pick any team and get a full head-to-head breakdown: points, race results, qualifying, average finish

## Stack

| Layer | Choice | Why |
|---|---|---|
| Framework | [Dash](https://dash.plotly.com/) | Real interactive web app, not a data science toy |
| Charts | Plotly | Beautiful out of the box, zero config |
| Styling | Dash Bootstrap + custom CSS | Dark F1 theme without fighting a design system |
| Data | [Jolpica API](https://api.jolpi.ca/) | Free Ergast replacement, no auth needed |
| Hosting | [Render](https://render.com/) | Zero-config deploys from GitHub |

## Run locally

```bash
git clone https://github.com/alonsoruizv/f1-dashboard.git
cd f1-dashboard

python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open [http://127.0.0.1:8050](http://127.0.0.1:8050)

## Deploy

Connected to Render via `render.yaml` — push to `main` and it ships automatically.
